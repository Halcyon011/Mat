# Степенные ряды — задача 1

Найти область сходимости ряда:

```
\sum_{n=0}^{\infty} \frac{2^n x^n}{(2n+1)^3}
```

Решение простыми шагами.

1) Общий член:  
``a_n(x) = (2^n x^n)/(2n+1)^3 = ((2x)^n)/(2n+1)^3``.

2) Радиус сходимости (корневой тест).  
``lim_{n→∞} |a_n|^{1/n} = |2x| * ((2n+1)^3)^{-1/n} → |2x|`` (так как ``(2n+1)^{3/n} → 1``).  
Условие сходимости: ``|2x| < 1`` ⇒ ``|x| < 1/2``. Значит, радиус ``R = 1/2``.

3) Концы интервала.  
- При ``x = 1/2``: ``a_n = 1/(2n+1)^3`` — сходится (часть p-ряда с p = 3 > 1).  
- При ``x = -1/2``: ``a_n = (-1)^n/(2n+1)^3`` — сходится абсолютно, т. к. ``\sum 1/(2n+1)^3`` сходится.

Итог: интервал (область) сходимости ``[-1/2, 1/2]``, сходимость абсолютная.

Примечание: сумма ряда определяет аналитическую функцию при ``|x| < 1/2``.
